Preguntas teóricas de AJAX con jQuery:

  • El método ajaxSetup() establece los valores por defecto en las peticiones Ajax. Indica por qué su uso no está recomendado: Porque su modificación de opciones es global, es decir, afecta a todas las invocaciones de AJAX.
  • Indica los 6 manejadores de eventos globales de AJAX.
    1. ajaxComplete()
    2. ajaxError()
    3. ajaxSend()
    4. ajaxStart()
    5. ajaxStop()
    6. ajaxSuccess()
  • Indica al objeto al que han de estar asociados los métodos anteriores: Tiene que estar asociados a un objeto jQuery del document
  • Los métodos AJAX abreviados son 6. Indica cuáles son y utilidad de cada uno.
    1. get(): Carga los datos del servidor usando HTTP con petición GET.
    2. getJSON(): Carga los datos en un JSON del servidor usando HTTP con petición GET
    3. getScript(): Carga un archivo JavaScript del servidor usando HTTP con petición GET y a continuación lo ejecuta.
    4. post(): Carga datos del servidor usando HTTP con petición POST.
    5. load(): Carga datos del servidor y devuelve el HTML introducido al elemento asignado.
  • En jQuery, indica la diferencia entre .ajaxSucess() y .ajaxComplete()
    1. ajaxSuccess(): Solo se llama cuando el servidor responde con un código 200 (“OK”), es decir cuando todo esta bien.
    2. ajaxComplete(): Siempre es llamado, sin importar si la petición Ajax ha finalizado con éxito o no.
